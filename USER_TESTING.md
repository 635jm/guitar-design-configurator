# User Testing Guide

Use this script to test the Guitar Design Configurator MVP in a browser.

## Before You Start

- Projects are saved only in the current browser.
- Clearing browser storage or switching devices/browsers may hide or remove projects.
- No login, backend sync, payment, 3D renderer, or AI image generation API is included yet.

## Test Script

1. Open the app.
2. Create a new project.
3. Choose a starter preset.
4. Open the configurator.
5. Apply a different preset.
6. Customize at least three options, such as finish, pickups, bridge, or pickguard.
7. Save the project.
8. Refresh the page and confirm the saved choices are still there.
9. Return to the dashboard.
10. Search for the project by name, body shape, or pickups.
11. Open the summary page.
12. Copy the summary.
13. Download the TXT summary.
14. Copy the AI image prompt.
15. Export the project JSON.
16. Return to the dashboard and export all projects JSON.

## Feedback To Capture

- What felt confusing?
- Did anything fail to save or reload?
- Were any buttons hard to find?
- Did the preview match the choices well enough for an MVP?
- Did the exported files download as expected?
- What would make this more useful for a real guitar design workflow?
